names_short = ['alc', 'mal', 'ash', 'alcash', 'mag', 'phe', 'fla', 'nfla', 'proant', 'col', 'hue', 'dil', 'prol']
plot_tree(clf, feature_names = names_short, filled=True);

Comparison to LDA:\ In LDA, the bkj coefficients for the proline feature were large and positive for class 0, and large and negative for both class 1 and 2. This indicates that this feature is very discriminative to separate class 0 from classes 1 and 2. This feature is used here for the first split of the tree. Proline<755 corresponds to mostly examples of classes 1 and 2, and Proline>755 corresponds to mostly examples of class 0. Other variables with large coefficients in LDA are flavanoid, color intensity, diluted wine. These variables are used as split variables in the second and third layer of the tree.

nfold = 5 # number of folds for cross-validation
max_leaf = 12
nb_leaf_all = np.arange(2,max_leaf+1)
risk_cv = np.zeros(shape=nb_leaf_all.shape)
for (i, nlea) in enumerate(nb_leaf_all):
    clf = DecisionTreeClassifier(max_leaf_nodes = nlea)
    risk_cv[i] = 1-(cross_val_score(clf, X,y, cv = nfold)).mean()
plt.figure()
plt.plot(nb_leaf_all, risk_cv)
plt.xlabel('Number of leaf nodes')
plt.ylabel('X-validation risk');
In [7]:
# Plot the best tree
best_nleaf = np.argmin(risk_cv)
clf = DecisionTreeClassifier(max_leaf_nodes = nb_leaf_all[best_nleaf])
clf.fit(X,y)
plot_tree(clf,  filled=True, feature_names = names_short);

In Scikit-learn the variable importance is measured via the average decrease in Gini impurity. Note that we could use alternatively the decrease in the accuracy (not implemented in Scikit-learn). The resulting feature importances align well with the results obtained with LDA. Features with large weights in LDA correspond to features with large feature importance (prol, col, fla, dil).
